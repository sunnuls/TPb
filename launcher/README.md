# HIVE Launcher

PyQt6 control panel for the HIVE multi-bot poker network.

## Features

- Account and window / emulator binding
- Bot pool start/stop and profiles
- Lobby scan and team seating
- Team card sharing and coordinated decisions
- Logs, settings, live view

## Quick start

```bash
pip install -e ".[dev,live]"
python -m launcher.main
```

Or: `START_LAUNCHER.bat`
