# roadmap14_full_hive_app.md
ФИНАЛЬНЫЙ РОУДМАП — Полноценное рабочее приложение HIVE Launcher

Цель: Создать удобное, логичное, без багов standalone-приложение (на базе уже существующего кода), в котором пользователь может:

- Добавлять аккаунты (с именем/ником)
- Выбирать/захватывать окно (браузер или клиент)
- Автоматически или вручную настраивать ROI (с live-превью)
- Запускать любое количество ботов
- Видеть список всех активных ботов (ник + номер + статус)
- Настраивать агрессию и параметры
- Получать live-вывод того, что каждый бот видит (скриншот + наложенные зоны и распознанные элементы)
- Автоматически сканировать лобби, находить столы с 1–3 игроками, заполнять их 3 ботами
- Активировать коллюзию (обмен картами) и 3vs1 манипуляции

Всё на базе уже существующего кода проекта (launcher, bridge, vision, central_hub, collective_decision, hive и т.д.). Никаких новых больших фреймворков.

## Фаза 1 — Основное окно лаунчера (UI/UX)
- Главное окно на PyQt6 + customtkinter (уже используется)
- Вкладки: Accounts | Live View | Bot Control | Settings | Logs
- Красивый, понятный интерфейс (тёмная тема)

## Фаза 2 — Управление аккаунтами + привязка
- Таблица аккаунтов с колонками: №, Ник, Окно, ROI статус, Статус бота
- Кнопка "Добавить аккаунт" + привязка к окну

## Фаза 3 — Захват окна + Авто + GUI ROI
- Кнопка "Выбрать окно" (авто-поиск + ручной выбор)
- Кнопка "Настроить ROI" — открывает удобный оверлей с рисованием мышкой (один раз)
- Live превью того, что видит бот (скриншот + выделенные зоны + распознанные карты/числа)

## Фаза 4 — Запуск и контроль ботов
- Спинбокс "Количество ботов"
- Большая кнопка START / STOP ALL
- Реал-тайм таблица активных ботов (ник + № + статус + текущий стол + стек + collective edge)
- Кнопка "Live View" для каждого бота (отдельное окно с тем, что он видит)

## Фаза 5 — Автоматизация HIVE (скан → 3vs1)
- Автоматический поиск столов с 1–3 игроками
- Авто-заход 3 ботов
- Авто-активация коллюзии (обмен картами)
- 3vs1 манипуляции

## Фаза 6 — Настройки и безопасность
- Глобальные и per-bot настройки (агрессия, equity threshold, delays и т.д.)
- Пресеты (Balanced, Aggressive, GodMode)
- Emergency STOP + авто-стоп при ошибках

## Фаза 7 — Финальная полировка и тесты
- Полная стабильность, отсутствие багов
- Красивый, логичный интерфейс
- Тесты на 10+ ботов одновременно
