# roadmap7_hive_vs_human_tournament.md — Автоматизированная игра ботов против людей (с предупреждением)

Цель: на своей платформе реализовать режим, где 3 координирующихся бота:
- сканируют столы
- находят выгодные (1–3 человека + места ≥3)
- садятся втроём
- обмениваются hole cards в реальном времени
- манипулируют против человека (3vs1)

Игроку **явно сообщается** о формате перед входом.

## Фаза 1 – Боты-сканировщики и авто-заполнение столов (2 недели)
- Взять table_selection.py и opportunity_detector.py
- Адаптировать под реальное лобби платформы (API или vision)
- Логика: если human_count ≤ 3 && seats_left ≥ 3 → 3 бота из пула получают команду join
- Приоритет: столы с 1 человеком + много свободных мест
- Добавить пул из 100+ ботов (multiprocessing / Docker)

## Фаза 2 – Реальная координация и обмен картами (1–2 недели)
- Расширить external_hub_client.py: обмен hole cards только между 3 ботами в одной сессии
- Добавить в collective_decision.py режим full_collusion: если 3 бота → equity с известными картами
- Создать collusion_activation.py: после посадки всех 3 — активация режима

## Фаза 3 – Манипуляции 3vs1 в реальном времени (2 недели)
- Усилить manipulation_logic.py:
  - collective equity > 65% → large bet/raise/all-in
  - < 40% → фолд / min call друг другу
  - Боты не блефуют друг против друга
- Интегрировать с real_executor.py (unsafe клики только при --unsafe)
