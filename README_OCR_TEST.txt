ШАГ 1: Подготовьте файлы
─────────────────────────
  1. Скопируйте в папку проекта:
     📄 poker_table_config (1).yaml  (скачанный из браузера)
     📄 photo_2026-01-16_19-35-24.jpg (ваш скриншот)

ШАГ 2: Установите Tesseract OCR
────────────────────────────────
  1. Скачайте: https://github.com/UB-Mannheim/tesseract/wiki
  2. Установите в: C:\Program Files\Tesseract-OCR
  3. Перезапустите терминал

  Проверка:
    tesseract --version

ШАГ 3: Запустите тест
─────────────────────
  Вариант А (автоматически):
    quick_ocr_test.bat

  Вариант Б (вручную):
    .\.venv\Scripts\Activate.ps1
    pip install pillow pytesseract pyyaml
    python test_real_ocr.py "photo_2026-01-16_19-35-24.jpg" "poker_table_config (1).yaml"

ШАГ 4: Проверьте результаты
────────────────────────────
  ✅ В консоли увидите распознанные карты
  ✅ Откройте zones_visualization.png - проверьте расположение зон


═══════════════════════════════════════════════════════════════
  🔧 ЕСЛИ КАРТЫ РАСПОЗНАЮТСЯ НЕПРАВИЛЬНО
═══════════════════════════════════════════════════════════════

1. Откройте zones_visualization.png
2. Если зоны сдвинуты:
   → Вернитесь в poker_overlay_advanced.html
   → Перенастройте зоны точнее
   → Скачайте новый конфиг
   → Повторите тест

3. Если зоны правильные, но текст неточный:
   → Используйте скриншот с высоким разрешением
   → Убедитесь, что карты четкие и крупные
   → Попробуйте классический вид карт в покер-клиенте
